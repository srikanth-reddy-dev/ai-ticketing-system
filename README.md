# AI Ticketing System

## Overview
This project is a simple AI-powered ticketing system that automatically categorizes and prioritizes user issues.

## Features
- Create support tickets
- Automatic categorization (IT, HR, Finance)
- Priority detection (High, Medium, Low)
- Real-time ticket display
- Clean UI with priority highlighting

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Database: SQLite
- AI Logic: Rule-based classification

## How It Works
1. User submits an issue
2. System analyzes keywords
3. Assigns category and priority
4. Displays ticket in dashboard
